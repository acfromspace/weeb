## Install

```
npm i @acfromspace/weeb

OR

yarn add @acfromspace/weeb
```

## Usage

```js
const weeb = require("@acfromspace/weeb");

weeb("what are you doing?!");
//=> "NANI are you doing?!"

weeb(420);
//=> Uncaught TypeError: これは文字列ではありません!
```
